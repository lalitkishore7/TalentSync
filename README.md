# TalentSync

VerifyHire is a trusted student job and internship platform focused on verified hiring, resume-based recommendations, and career awareness.

## Frontend Scope

This repository will be used for the frontend implementation of the TalentSync project.

## Core Product Vision

- Verified companies only
- Trusted jobs and internships for students
- AI-powered resume matching
- Real-time IT industry news feed
- Fraud prevention focused experience

## Tagline

Verified Companies. Trusted Opportunities. Smarter Careers.
